前端训练营 00学习方法与构建知识体系

2020-05-02

思考问题

  • 关于前端，你都会什么？
    • 如果让你写一本写关于前端的书，目录会写什么？
    • 如果没有知识体系，学再多的东西，只是在垃圾堆上堆东西
  • 关于前端你都不会什么？

前端技能模型

  • 编程能力
    • 把复杂的逻辑 理清楚 分拆出来 最后实现 这种能力叫编程能力
    • 解决难写不出来的问题
  • 架构能力
    • 对复杂系统进行分析 对复杂系统所需要的程序 进行软件设计以及对多个程序之间的协同管理叫架构能力
    • 解决大写不出来的能力
    • 带着目的读源码
  • 工程能力
    • 解决人多 怎么协作的问题
    • 比如 有的外行会按照 代码行数 或者 出 bug 数的数量来管理
  • 领域知识
    • html css js 浏览器 api ，后端最痛苦的是 css 因为领域知识相差比较远

什么是好的面试题

  • 区分度
    • 基础差的人能说几句
    • 中等人 能回答出中等水平
    • 高级的能回答出高级水平
  • 覆盖面
  • 深度
